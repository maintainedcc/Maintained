## Usage

Use Maintained badges to give quick live status updates and fancy badges for your projects.

<img width="100%" alt="image" src="https://user-images.githubusercontent.com/35588499/175190275-63ed482e-b1bc-4f19-84c3-b1a0ab2ec52f.png">

- Quick sign in with GitHub: no additional accounts needed
- Edit badge content from the dashboard and have it update in real time
- Create badges with brand or Bootstrap icons
- Create badges with more than two fields, for badges with many icons
- Make any existing plaintext API endpoint into a badge
